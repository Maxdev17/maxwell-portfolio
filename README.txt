## Portfolio (ready)
Files:
- index.html
- styles.css
- script.js

To deploy on GitHub Pages:
1. Create a new GitHub repository (e.g. `maxwell-portfolio`)
2. Push these files to the repository's main branch.
3. In the repository settings -> Pages, choose branch `main` (or `gh-pages`) and root, then save.
4. Your site will be available at `https://<username>.github.io/<repo>/` (or at a custom domain if you buy one).
